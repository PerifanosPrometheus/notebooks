--- layout: home search_exclude: true image: images/logo.png --- This site is a place to host Jupyter Notebooks for the [Neurodata](https://neurodata.io/) lab at Johns Hopkins University. If you are a Neurodata member with write access and would like to add a notebook, check out the [Github repo](https://github.com/neurodata/notebooks) that builds this site. Simply upload your notebook to the [_notebooks](https://github.com/neurodata/notebooks/tree/master/_notebooks) folder, following all of the instructions there. For more information on the lab, visit our [about page](https://neurodata.io/about/), or our [Github](https://github.com/neurodata) to see other projects and open source software. This site is built with [fastpages](https://github.com/fastai/fastpages), An easy to use blogging platform with extra features for Jupyter Notebooks. [fastpages](https://github.com/fastai/fastpages) automates the process of creating blog posts via GitHub Actions, so you don't have to fuss with conversion scripts. A full list of features can be found on [GitHub](https://github.com/fastai/fastpages). # Posts
